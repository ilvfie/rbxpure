## Installation
Using Roblox Studio's command bar:
```lua
game:GetService("HttpService").HttpEnabled = true
loadstring(game:GetService("HttpService"):GetAsync("https://raw.githubusercontent.com/ilvfie/rbxpure/master/install.lua"))()
```
